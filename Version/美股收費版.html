                # 連結

                # 功能

                功能同台股免費版

                不同的是

                1. 取價改為美股網站
                2. 整份試算表的幣別為「美金」
                3. 儀表板 - 新增可查詢【道瓊指數】【那斯達克】💡【費半指數】【美股個股】的簡易資料
                4. 儀表板新增美金兌台幣的匯率
                5. 股票代號採手動自行更新
                6. 平倉功能 進階功能

                💡【費半指數】> 此指數目前google方的圖表資料有誤，只有下面數字。沒有圖表

                其他基本功能

                # 如何使用

                1. 購買 - 參考 進階功能 > 購買 > 付款方式
                2. 啟用 - 參考 進階功能 > 購買 > 付款後開通
