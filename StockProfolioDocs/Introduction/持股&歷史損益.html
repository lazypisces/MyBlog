            # 持股&歷史損益

            將從【交易紀錄】中的紀錄篩選出持有/已脫手的股票，並做計算

            • 儀表板的損益圖，是根據這兩頁的資料轉換而來
            • 篩選部分，點【儀表板】的更新交易紀錄按鈕即可，這裡不需要任何操作
            • 根據版本不同，篩選的方式也不同，如下表
            版本持股分頁歷史損益
            台股免費版持股數不 = 0持股數 = 0
            台股收費版未平倉已平倉

            差別: 若單一個股交易次數多，不容易判別之後的交易是否有確實獲利，會和前之前的獲利一起被平均

            若此分頁有出現錯誤，很可能是【交易紀錄】的紀錄有錯誤，導致計算公式無法正確計算
